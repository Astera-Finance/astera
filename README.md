## Setup
**To begin testing:**
```
1. run 'npm i' in project root
2. remove '.example' suffix from .env.example and hardhat.config.ts.example
3. run 'yarn compile' or 'npm run compile'
4. run 'yarn tets' or 'npm run test'
```
